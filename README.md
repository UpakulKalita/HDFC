# Insurance Planner (Flutter)

A Flutter-based frontend for the Insurance Planner application.

## Getting Started

1.  **Install Dependencies**:
    ```bash
    flutter pub get
    ```

2.  **Run the App**:
    - Web: `flutter run -d chrome`
    - Windows: `flutter run -d windows`

## Features
- Login Page (Implemented)
- Customer Details (Planned)
